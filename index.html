Puzzle 1

Fun fact: 11 is the smallest strobogrammatic prime.
A prime number is only divisible by 1 and itself.
A strobogrammatic number is one that looks the same when rotated 180° (turned upside down).
For example, 69 and 8118 are strobogrammatic.

The puzzle: What is the next smallest strobogrammatic prime after 11?

Extension: Are there infinitely many strobogrammatic primes?

Show full solution

Solution: 101.
Extension solution: It is currently unknown whether there are infinitely many strobogrammatic primes (an open problem).

Reasoning: Strobogrammatic digits are 0,1,6,8,9 (with 6↔9). The next candidate after 11 is 101, which is prime.

Puzzle 2

Fun fact: A palindrome is a number that reads the same forwards and backwards.
Any palindrome with an even number of digits is always divisible by 11.
For example:

  • 1221 ÷ 11 = 111
  • 3773 ÷ 11 = 343
  • 985589 ÷ 11 = 8969

The puzzle: Find a 4-digit palindrome that is divisible by 11².

Extension: How many 4-digit palindromes are divisible by 11²?

Show full solution

Solution: any number from this set: {1331, 2662, 3993, 4114, 5445, 6776, 8228, 9559}

Extension solution: 8

Reasoning: Any 4-digit palindrome has the form abba, whose value is 1000a+100b+10b+a = 1001a+110b = 11(91a+10b).
Divisibility by 121 requires 91a+10b to be a multiple of 11. Working modulo 11, this is 3a − b ≡ 0, so b ≡ 3a (mod 11).
With 1 ≤ a ≤ 9 and 0 ≤ b ≤ 9, the valid (a,b) pairs are: (1,3), (2,6), (3,9), (4,1), (5,4), (6,7), (8,2), (9,5).
Thus the 8 palindromes divisible by 121 are:
1331, 2662, 3993, 4114, 5445, 6776, 8228, 9559.

Puzzle 3

Fun fact: According to the Gauss–Wantzel Theorem, a regular hendecagon (an 11-sided polygon) cannot be constructed using only a compass and straightedge.

The puzzle: How many diagonals does a hendecagon have?

Extension: If all diagonals are drawn inside a regular hendecagon, how many distinct regions are created?

Show full solution

Solution: 44

Extension solution: 375

Reasoning:

Diagonals: In an n-gon, diagonals = n(n−3)/2. For n=11: 11×8/2 = 44.

Regions with all diagonals: With no three diagonals concurrent in the interior (true for a regular polygon), the number of regions formed inside the 11-gon is:
C(11,4) + C(11,2) + 1 − 11 = 330 + 55 + 1 − 11 = 375.

(Here C(n,k) counts intersection points of diagonals: C(11,4) interior crossings and C(11,2) edges/diagonals, and we subtract 11 to exclude the exterior regions adjacent to the sides.)
